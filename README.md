# tetris_score_server_frontend

以下サーバのフロントエンドに関する検討用として作成(仮)

https://github.com/seigot/tetris_score_server  
https://github.com/seigot/tetris_battle_server  

## 環境構築
### docker engineのインストール
[dockerインストール](https://docs.docker.jp/engine/installation/index.html)を参照

### docker composeのインストール
[docker compoerのインストール](https://docs.docker.jp/compose/install.html#compose)を参照

### docker composeの実行
`/tetris_score_server_frontend`に移動して
```
docker-compose up
```
を実行する。
